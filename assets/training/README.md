# Training target sprites

Both PNG files are 48 x 24 pixel sprite sheets with two 24 x 24 frames arranged left to right.

- `straw-dummy.png`: stationary one-hit target used by random spawn training.
- `moving-target.png`: durable bullseye used by random movement training.

Run `npm run assets:training` to restore the generated placeholders and `training-cheer.wav`.
